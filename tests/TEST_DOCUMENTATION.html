2. White Box Testing

White box testing examines internal code structure, branches, and logic paths. The tester has full knowledge of the source code. We test the generate_password() and authenticate_user() functions in app/services/auth_service.py.

2.1 Module Under Test: generate_password()

def generate_password(length=10): chars = string.ascii_letters + string.digits # safe alphanumeric only while True: pwd = ''.join(secrets.choice(chars) for _ in range(length)) if (any(c.isupper() for c in pwd) and # ← Branch A any(c.islower() for c in pwd) and # ← Branch B any(c.isdigit() for c in pwd)): # ← Branch C return pwd

4. How to Run Tests

Prerequisites

# Install test dependencies pip install pytest pytest-cov # Navigate to project root cd medicore-hms

Run Full Test Suite

# Option 1: Run directly python tests/test_suite.py # Option 2: Run with pytest (more detail) python -m pytest tests/test_suite.py -v # Option 3: Run with coverage report python -m pytest tests/test_suite.py -v --cov=app --cov-report=term-missing # Option 4: Run only White Box tests python -m pytest tests/test_suite.py -v -k "WhiteBox" # Option 5: Run only Black Box tests python -m pytest tests/test_suite.py -v -k "BlackBox"
